    Staticset

    • set(picture: Picture, callback: (paint: Paint) => boolean): void
      Beta

      Traverse all descendant paint nodes of a picture and invoke a callback on each. The callback receives the correctly typed Paint subclass (Shape, Scene, Picture, or Text). Return false from the callback to stop traversal early.

      Parameters

      • picture: Picture

        The root picture to traverse

      • callback: (paint: Paint) => boolean

        Called for each descendant paint. Return false to stop.

      Returns void

      Example

      Accessor.set(picture, (paint) => {
        if (paint instanceof TVG.Shape) {
          paint.fill(0, 0, 255);
        }
        return true; // continue traversal
      });
