    Interface FontProviderBeta

    Interface for pluggable font sources.

    A font provider resolves a font name into binary font data. Implement this interface to load fonts from any source — a self-hosted CDN, a local server, or any custom storage.

    Example

    TVG.Font.provider({
      fetch: async (name) => {
        const res = await fetch(`/my-fonts/${name}.ttf`);
        return { data: new Uint8Array(await res.arrayBuffer()), type: 'ttf' };
      }
    });

    await TVG.Font.load('my-font');
