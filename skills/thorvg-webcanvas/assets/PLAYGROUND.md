## Playground Examples

When the user asks "how do I do X with webcanvas?" and you need a working reference, check
the playground. It contains 49 interactive examples with live code and preview. Each example
demonstrates a specific API pattern and can be used as a starting template.

URL pattern: `https://thorvg-playground.vercel.app/showcase/{id}`

| Category | Examples |
|----------|----------|
| **Basic** (20) | shapes, path, stroke, gradients, radial-gradient, fill-rule, fill-spread, gradient-stroke, gradient-transform, opacity, scene-transform, scene-blending, custom-transform, direct-update, transform-animation, trim-path, image-rotation, image-scaling, stroke-line, stroke-miterlimit |
| **Advanced** (13) | accessor, blending, bounding-box, clipping, duplicate, masking, masking-methods, gradient-masking, scene-effects, effect-drop-shadow, error-handling, intersects, viewport |
| **Text** (4) | text-static, text-layout, text-line-wrap, text-animation |
| **Media** (7) | picture-svg, picture-png, picture-jpg, picture-webp, picture-raw, lottie-animation, lottie-expressions |
