    Enumeration BlendMethod

    Blend method for compositing paint layers.

    Defines various blending modes for combining a source paint (top layer) with a destination (bottom layer). Notation: S = source paint, D = destination, Sa = source alpha, Da = destination alpha.

    Normal: 0

    Perform alpha blending (default). S if (Sa == 255), otherwise (Sa * S) + (255 - Sa) * D
